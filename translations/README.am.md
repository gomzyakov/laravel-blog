## рЅЦрѕјрїЇ рЅаріарїѕрѕГ рІЇрѕхрїЦ ріЦріЋрІ┤рЅх ріЦріЋрІ░рѕџрѕ░рѕФ?


ріарѕхрЅђрІхрѕўрѕЁ Docker ріЦріЋрІ░рЅ░рїФріљрѕЁ ріарѕЮріЊрѕѕрѕЂрЇб ріФрѕЇрѕєріљрЇБ рѕЇріГ рЅа[Mac](https://docs.docker.com/desktop/install/mac-install/)рЇБ [Windows](https://docs.docker.com/desktop/install/windows) рѕІрІГ рІФрІхрѕГрїЅрЅхрЇб -install/) рІѕрІГрѕЮ [Linux](https://docs.docker.com/desktop/install/linux-install/)рЇб
